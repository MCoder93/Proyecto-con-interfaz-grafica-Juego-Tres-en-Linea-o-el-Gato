Proyecto con interfaz gráfica en Python: Juego Tres en Línea o También llamado el Gato.

Este proyecto recrea el clásico juego "Tres en Línea", conocido en Chile como "El Gato". 
Fue desarrollado como parte de mi proceso de aprendizaje y consolidación en programación con Python, 
aplicando lógica de juego, estructuras condicionales y control de flujo. Más allá de su funcionalidad, 
este proyecto refleja mi compromiso con el desarrollo de soluciones educativas y accesibles. 
Está pensado como una herramienta motivadora para quienes están comenzando en programación, 
demostrando que es posible crear experiencias interactivas con código claro y bien estructurado.

Objetivos del proyecto:
- Reforzar habilidades en Python mediante un enfoque práctico.
- Promover el aprendizaje de programación a través de ejemplos simples y funcionales.
- Sentar las bases para futuras mejoras, como interfaces gráficas o modos multijugador.

Gracias por visitar este proyecto. Espero que sirva como inspiración para quienes están comenzando 
en el mundo de la programación. Cada línea de código es una oportunidad para aprender, crear y compartir.
